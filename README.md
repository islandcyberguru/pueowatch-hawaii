# 🦉 PueoWatch Hawaiʻi

**Free community cybersecurity alerts and digital safety resources for Hawaiʻi residents.**

A nonprofit community platform by [Hawaiian Islands CyberGuru Academy](https://cyberguruacademy.org).

## Features
- 🚨 Live scam alerts — reported by community, filtered by island
- 📖 Plain-language digital safety guides (no tech background needed)
- 📍 Local resource directory with Hawaii-specific organizations
- 🦉 Pueo-guided onboarding in the spirit of aloha

## Tech Stack
- React 18
- Deployed via Cloudflare Pages
- Hosted on islandcybersecure.com

## Development

```bash
npm install
npm start
```

## Deployment
Deployed automatically via Cloudflare Pages on push to `main`.
